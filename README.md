# customer_behaviour_analysis
Data analysis project using SQL for querying, Python for data cleaning, and Power BI for visualization with real-world datasets

## üìå Overview
This project implements an **end-to-end data analytics pipeline** to analyze customer shopping behavior and purchasing trends. The primary objective is to transform raw transactional data into **actionable business insights** that can guide marketing strategy, customer segmentation, and demand forecasting.  
The findings are delivered through statistical analysis, optimized SQL queries, and a **dynamic Power BI dashboard** designed with strong emphasis on clarity and user experience.

---

## üìä Business Problem
Understanding customer purchase behavior is critical for improving:
- **Customer retention & segmentation**
- **Revenue contribution analysis**
- **Seasonal demand planning**
- **Category-wise purchasing trends**
- **Data-driven decision making**

A detailed problem statement and domain context are available in the supporting documents.

## üîç Methodology & Key Steps
1. **Dataset Loading**  
   Imported into Python using Pandas for structured processing.
2. **Exploratory Data Analysis (EDA)**  
   Evaluated distributions, correlations, and behavioral trends using Matplotlib and statistical summaries.
3. **Data Cleaning & Preprocessing**  
   Handled missing values, inconsistencies, and structured data formatting.
4. **SQL-Driven Analytics**  
   Executed optimized PostgreSQL queries using:  
   `customer_behavior_sql_queries.sql`
5. **Dashboard Design & Development**  
   Built an **interactive Power BI dashboard**:  
   `customer_behavior_dashboard.pbix`
6. **Results Documentation**  
   Insights summarized in a structured analytical report and stakeholder-ready presentation using Gamma.

## üß† Key Findings (Summary)
- A minority customer segment contributes to the majority of total revenue (Pareto pattern observed).
- Strong seasonal trends detected in specific product categories.
- Customer spending clusters identified, enabling meaningful segmentation.
- Purchase frequency correlates with higher retention potential.
- Insights provide strategic input for **targeted marketing and demand planning**.
